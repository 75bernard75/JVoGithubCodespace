### Step 3: Copy-Paste Server Setup

**SSH to server as root, then copy-paste this entire block**:

```bash
# Update system
apt update && apt upgrade -y

# Install software
apt install -y curl wget git nginx certbot python3-certbot-nginx htop rsync

# Create deploy user
useradd -m -s /bin/bash deploy
usermod -aG sudo deploy

# Create website directory
mkdir -p /var/www/jvogaming
chown deploy:deploy /var/www/jvogaming
chmod 755 /var/www/jvogaming

# Create logs directory
mkdir -p /var/log/jvogaming
chown deploy:deploy /var/log/jvogaming
chmod 755 /var/log/jvogaming

echo "‚úÖ Server setup complete!"
```

**Wait**: 3-5 minutes for packages to install

### Step 4: Configure nginx

**Still SSH to server as root**:

```bash
# Create config file
cat > /etc/nginx/sites-available/jvogaming << 'EOF'
server {
    listen 80;
    server_name _;
    return 301 https://$host$request_uri;
}

server {
    listen 443 ssl http2;
    server_name _;
    
    ssl_certificate /etc/letsencrypt/live/jvogaming/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/jvogaming/privkey.pem;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    
    root /var/www/jvogaming;
    index index.html;
    
    add_header Strict-Transport-Security "max-age=31536000" always;
    
    location ~* \.(js|css|png|jpg|gif|svg|woff2)$ {
        expires 30d;
    }
    
    location / {
        try_files $uri $uri/ /index.html;
    }
}
EOF

# Enable site
ln -s /etc/nginx/sites-available/jvogaming /etc/nginx/sites-enabled/

# Remove default
rm /etc/nginx/sites-enabled/default 2>/dev/null || true

# Test config
nginx -t

# Start nginx
systemctl enable nginx
systemctl start nginx

echo "‚úÖ nginx configured!"
```

### Step 5: Get SSL Certificate

**Still as root**:

```bash
# Create temp website to verify domain
mkdir -p /var/www/html
echo "placeholder" > /var/www/html/index.html

# Get certificate (interactive)
certbot certonly --standalone \
  -d jvogaming.com \
  -d www.jvogaming.com \
  --email your@email.com \
  --agree-tos \
  --no-eff-email

# Note: Replace jvogaming.com with your actual domain!
```

**When prompted**:
- "Please enter the domain name(s)..." ‚Üí Enter your domain
- Accept terms ‚Üí Type: Yes

**Wait**: Certificate is issued (1-2 minutes)

### Step 6: Setup SSH Key for GitHub

**Switch to deploy user**:

```bash
su - deploy
```

**Generate SSH key**:

```bash
# Generate key
ssh-keygen -t ed25519 -C "github-actions" -f ~/.ssh/github-deploy -N ""

# Display private key (COPY THIS ENTIRE OUTPUT)
cat ~/.ssh/github-deploy

# Add to authorized_keys (for deployment)
cat ~/.ssh/github-deploy.pub >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
```

**Copy the private key** (everything from `-----BEGIN` to `-----END`) ‚úÖ

### Step 7: Setup Sudo for Deploy User

**Back as root**:

```bash
visudo
```

**At the end of file, add**:

```
deploy ALL=(ALL) NOPASSWD: /usr/bin/systemctl
```

**Save**: Ctrl+X ‚Üí y ‚Üí Enter

### Step 8: Add GitHub Secrets

**Go to**: https://github.com/75bernard75/JVoGithubCodespace/settings/secrets/actions

**Add 4 secrets**:

| Name | Value |
|------|-------|
| DEPLOY_HOST | jvogaming.com (or your IP) |
| DEPLOY_USER | deploy |
| DEPLOY_PATH | /var/www/jvogaming |
| DEPLOY_KEY | [Paste entire private key] |

**‚ö†Ô∏è IMPORTANT**: DEPLOY_KEY must include BEGIN/END lines!
