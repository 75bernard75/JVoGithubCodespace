## 4. PROCESSUS D'OPTIMISATION IMAGES

### 4.1 Pipeline de Compression

```mermaid
Original Image (4000x3000, 8-12MB)
    ↓
Redimensionner → 1200x800px
    ↓
Convertir en WebP → 75% réduction
    ↓
Générer JPEG fallback → 85% réduction
    ↓
Générer thumbnails → 150x150px (20% de l'original)
    ↓
Tests performance (Lighthouse)
    ↓
Upload + CDN cache
```

### 4.2 Specs par Format

| Format | Source | Output | Compression | Use Case |
|--------|--------|--------|-------------|----------|
| **WebP** | PNG/JPG | 1200x800 | 75% | Modern browsers (90%+) |
| **JPEG** | Original | 1200x800 | 85% | Fallback IE/Safari |
| **WebP Thumb** | WebP | 300x300 | 60% | Card/List view |
| **JPEG Thumb** | JPEG | 300x300 | 65% | Card/List view |

### 4.3 Outil de Compression Recommandé

```bash
# ImageMagick (gratuit, ligne de commande)
convert input.jpg -quality 85 -resize 1200x800 output.jpg
convert input.jpg -define webp:method=6 output.webp

# Ou Squoosh CLI (Google)
npx @squoosh/cli --webp auto --mozjpeg auto image.jpg

# Batch processing
for file in *.jpg; do
  convert "$file" -quality 85 -resize 1200x800 "optimized/${file%.jpg}.jpg"
  convert "$file" -quality 90 -resize 1200x800 "optimized/${file%.jpg}.webp"
done
```

### 4.4 Stratégie Lazy Loading

```html
<!-- Image Standard avec lazy loading -->
<picture>
  <source srcset="images/ps5-hero-1200x800.webp" type="image/webp">
  <source srcset="images/ps5-hero-1200x800.jpg" type="image/jpeg">
  <img 
    src="images/ps5-hero-1200x800.jpg"
    alt="PlayStation 5 console noire - 4K gaming next-gen"
    loading="lazy"
    decoding="async"
    width="1200"
    height="800">
</picture>
```
